# 인증 & 계정 관리 PRD

## 1. 개요

- **Feature명**: 인증 & 계정 관리
- **소속 Milestone**: M-001 (MVP 핵심 기능)
- **한 줄 요약**: 글로벌 및 국내 유저 회원가입, 프로필, 탈퇴, 성인인증과 계정 전체 생애주기 관리

### 핵심 설계 결정

- **단일 회원 지위**: 작가/독자 구분 없음. 가입한 누구나 내 서점을 만들면 작가로 활동 가능
- **회원 구분**: 회원/비회원 구분 없이 단일 회원 지위. 비회원은 콘텐츠 열람만 가능 (구매/구독/팔로우 불가)
- **회원 유형**: 회원 / 한국 회원 / 글로벌 회원
- **서점 오픈**: 한국 유저만 MVP에서 가능 (작가 전환)
- **Passwordless**: 소셜로그인 + 전화번호 OTP만. 비밀번호 개념 없음
- **소셜 로그인 Provider**:
  - Korea: 카카오 / 네이버 / Google
  - Global: Google / Facebook / Line / 전화번호
- **국가 선택 UI**: 최초 로그인 화면에서 한국(🇰🇷) / 외국인(🌏) 버튼으로 국가 선택. 선택 후 해당 국가의 소셜 Provider가 표시됨
- **OTP 정책**:
  - OTP 유효시간: 3분
  - 최대 시도 횟수: 5회
  - SMS 쿨다운: 60초
  - 시간당 제한: 5회/시간, 10회/일 (전화번호 기준)
- **성인인증 정책**:
  - 성인 나이: 한국 만 19세, 글로벌 만 20세
  - 유효기간: 1년 (1년 경과 시 재인증)
  - 유효기간 내 재인증 불가 (버튼 비활성화)
  - 한국: NICE/KCB 통신사 본인확인 (계약 완료)
  - 글로벌: AWS Textract AnalyzeID (Bytes direct, S3 불필요)
- **탈퇴 정책**:
  - 탈퇴 요청 시 즉시 로그아웃
  - 30일 유예기간 중 재로그인 시 동의하면 계정 복구
  - 포스트 데이터: 익명 변환 (author → "탈퇴한 사용자")
  - 영구 삭제 대상: 계정 정보, 소셜 연동, 팔로우 관계
- **JWT 세션 전략**:
  - App: Refresh Token 180일 (6개월), Access Token 15분
  - Web: Refresh Token 7일, Access Token 15분

## 3. 유저스토리

- **US-1**: 회원으로서, 소셜 계정(한국: 카카오/네이버/Google, 글로벌: Google/Facebook/Line)으로 가입하고 싶다, 빠르게 가입하기 위해
- **US-2**: 회원으로서, 전화번호 + OTP로 가입/로그인하고 싶다, 소셜 계정 없이도 이용하기 위해
- **US-3**: 회원으로서, 프로필(닉네임, 이미지)을 설정/수정하고 싶다, 나만의 정체성을 만들기 위해
- **US-4**: 한국 회원으로서, 전화번호로 성인 인증하고 싶다, 성인 콘텐츠를 열람하기 위해
- **US-5**: 글로벌 회원으로서, 여권 OCR로 성인 인증하고 싶다, 성인 콘텐츠를 열람하기 위해
- **US-6**: 회원으로서, 회원 탈퇴를 요청하고 싶다, 서비스 이용을 중단하기 위해

## 4. 유저 플로우

### 메인 플로우

**Flow 1: 소셜 로그인 가입 (US-1)**

```
[앱 진입] → [로그인 화면] → [소셜 Provider 선택]
  → 한국: 카카오/네이버/Google
  → 글로벌: Google/Facebook/Line
→ [OAuth 인증] → [닉네임 입력 (최초)] → [가입 완료 → 홈]
```

**Flow 2: 전화번호 OTP 가입/로그인 (US-2)**

```
[가입] 전화번호 입력 → SMS 인증코드 → 닉네임 입력 → 가입 완료
[재로그인] 전화번호 입력 → SMS 인증코드 → 로그인 완료
```

**Flow 3: 프로필 설정 (US-3)**

```
[마이페이지] → [프로필 편집] → [닉네임/이미지 수정] → [저장] → [완료]
```

**Flow 4: 성인 인증 - 한국 (US-4)**

```
[성인 콘텐츠 접근 시도] → [성인 인증 필요 안내]
→ [전화번호 입력] → [SMS 인증] → [통신사 본인 확인] → [인증 완료]
```

**Flow 5: 성인 인증 - 글로벌 OCR (US-5)**

```
[성인 콘텐츠 접근 시도] → [성인 인증 필요 안내]
→ [여권 촬영/업로드] → [OCR 판독] → [생년월일 확인] → [인증 완료]
```

**Flow 6: 회원 탈퇴 (US-6)**

```
[설정] → [회원 탈퇴] → [탈퇴 사유 선택] → [탈퇴 요청 제출] → [즉시 로그아웃]
→ [유예기간 30일: 재로그인 시 "탈퇴 철회 동의" 팝업 → 동의 시 계정 복구]
→ [30일 경과: 계정 영구 삭제 + 포스트 익명 변환("탈퇴한 사용자")]
```

### 대안 플로우

- OAuth 실패 → 에러 안내 + 다른 Provider 제안
- SMS 미수신 → 재발송 (60초 쿨다운)
- OCR 판독 실패 → 재촬영 안내
- 이미 가입된 전화번호 → 기존 계정 로그인 유도
- 소셜 계정 연동 충돌 → 계정 병합 or 다른 방법 안내

## 7. 수용 기준 (Acceptance Criteria)

### AC-1 (US-1 관련: 소셜 최초 가입)

- **Given**: 미가입 사용자가 로그인 화면에 있을 때
- **When**: 소셜 버튼(카카오/Google 등)을 클릭하면
- **Then**: OAuth 인증 후 닉네임 설정 화면으로 이동하고 가입이 완료된다

### AC-2 (US-1 관련: 소셜 재로그인)

- **Given**: 이미 가입된 사용자가 소셜 로그인하면
- **When**: OAuth 인증이 성공하면
- **Then**: 닉네임 설정 없이 바로 홈으로 이동한다

### AC-3 (US-1 관련: 국가별 Provider 분기)

- **Given**: 한국 사용자가 로그인 화면에 있을 때
- **When**: 화면이 로드되면
- **Then**: 카카오/네이버/Google 버튼이 표시된다. 글로벌 사용자에게는 Google/Facebook/Line이 표시된다

### AC-4 (US-2 관련: OTP 발송)

- **Given**: 전화번호를 입력한 사용자가
- **When**: SMS 발송 버튼을 누르면
- **Then**: 인증코드가 발송되고 OTP 입력 화면으로 이동한다

### AC-5 (US-2 관련: OTP 오류)

- **Given**: OTP 입력 화면에서
- **When**: 잘못된 인증코드를 입력하면
- **Then**: 에러 메시지가 표시되고 재입력할 수 있다

### AC-6 (US-2 관련: OTP 재발송 쿨다운)

- **Given**: OTP 재발송을 요청한 후
- **When**: 60초가 경과하지 않았으면
- **Then**: 재발송 버튼이 비활성화되고 남은 시간이 표시된다

### AC-7 (US-3 관련: 프로필 수정)

- **Given**: 회원이 프로필 편집 화면에서
- **When**: 닉네임/이미지를 수정하고 저장하면
- **Then**: 변경 사항이 즉시 반영된다

### AC-8 (US-4 관련: 한국 성인인증)

- **Given**: 한국 회원이 성인 인증 화면에서
- **When**: 전화번호 + 통신사 본인 확인을 완료하면
- **Then**: 성인 인증이 승인되고 성인 콘텐츠 접근이 허용된다

### AC-9 (US-5 관련: 글로벌 성인인증 성공)

- **Given**: 글로벌 회원이 여권을 촬영/업로드하면
- **When**: OCR이 생년월일을 판독하고 한국 회원: 만 19세 이상, 글로벌 회원: 만 20세 이상이면
- **Then**: 성인 인증이 완료된다

### AC-10 (US-5 관련: OCR 실패)

- **Given**: OCR 판독이 실패하면
- **When**: 에러가 발생하면
- **Then**: 재촬영 안내가 표시되고 다시 시도할 수 있다

### AC-11 (US-6 관련: 회원 탈퇴)

- **Given**: 회원이 탈퇴 화면에서 사유를 선택하고
- **When**: 탈퇴 요청을 제출하면
- **Then**: 탈퇴 요청이 접수되고 즉시 로그아웃된다. 30일 유예기간 동안 재로그인 시 '탈퇴 철회' 동의 팝업이 표시되고, 동의하면 계정이 복구된다. 30일 경과 시 계정이 영구 삭제되고 포스트는 '탈퇴한 사용자'로 익명 변환된다

| #   | 상황                                  | 처리 방법                                                                                                                                                         | 스코프 |

| 2   | 전화번호로 가입 후 소셜 연동          | 설정에서 소셜 계정 추가 연동 가능                                                                                                                                 | 후속   |
| 3   | SMS 미수신 (국가별 통신사 문제)       | 재발송은 60초 쿨다운, 시간당 5회, 일일 10회 제한 + "소셜 로그인으로 전환" 안내                                                                                    | MVP    |
| 4   | OCR 판독 불가 여권 (훼손/화질)        | 재촬영 안내 + 3회 실패 시 고객센터 문의 유도                                                                                                                      | MVP    |

## 10. 우선순위 & 스코프

### MVP (이번 Feature)

- US-1: 소셜 로그인 가입 (한국 3사 + 글로벌 3사 + 전화번호)
- US-2: 전화번호 OTP 가입/로그인 (Passwordless)
- US-3: 프로필 설정/수정
- US-4: 한국 성인인증 (통신사 본인확인)
- US-5: 글로벌 성인인증 (AWS Textract AnalyzeID)
- US-6: 회원 탈퇴 (30일 유예기간 + 탈퇴 철회)
- JWT 세션 관리 (App 180일 / Web 7일)
- 엣지 케이스 #1, 3~11 처리

### 후속 개선

- 전화번호 가입 후 소셜 연동 (엣지 케이스 #2)
- OCR 비용 최적화: Tesseract.js 클라이언트 Tier 추가 검토
- NFC 칩 읽기를 통한 위변조 완전 방지 (네이티브 앱 필요)
- 2FA (이중 인증) 추가
- 소셜 계정 연동 관리 화면
