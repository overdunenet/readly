## 요약

### 핵심 인사이트

1. **서점(Bookstore) 컨셉 도입**: 별도의 "작가 전환"이 아니라, 모든 유저에게 "내 서재"(읽기)와 "내 서점"(쓰기/판매)이 부여됨. 서점 오픈 최소 요건: 필명 + 서점명
2. **전자책 뷰어 중심 읽기 경험**: 구매한 작품은 전자책 뷰어로 읽으며, 원문/번역 모드 전환, 이어서 읽기(페이지↔화차 동기화) 지원
3. **포인트 기반 결제 모델**: 포인트 충전 → 작품 구매 → 번역 언어 선택(추가금). 번역은 비용 센터가 아닌 수익 센터
4. **3계층 태그 시스템**: GL/BL/Hetero(고정) → 하위 장르 태그(사전 정의) → 작가 커스텀 태그(검색 매칭만)
5. **MVP = 웹 전체, 앱은 차후**: React 웹앱에서 전자책 뷰어 포함 모든 기능 제공. 네이티브 앱은 Growth 단계에서 별도 기획

### 주요 정책 결정

| 정책                  | 결정                                                    |
| --------------------- | ------------------------------------------------------- |
| MVP 서점(작가) 권한   | 한국 유저만                                             |
| 서점 비활성화         | 불가 (한 번 열면 닫을 수 없음)                          |
| 작가 콘텐츠 수정      | 허용, 로그 없음, 수정 알림은 작가 선택                  |
| 구매 작품 보장        | 작가 삭제해도 내 서재 아카이브 유지                     |
| 작가 콘텐츠 수정 반영 | 구매자 뷰어/내 서재에 실시간 반영                       |
| 성인 콘텐츠           | 작가가 등급 등록, 한국 유저는 전화번호 인증 필수        |
| 탈퇴                  | 메일 요청 방식 (간소화)                                 |
| 다중 서점             | MVP에서 불가 (1인 1서점)                                |
| UI 로컬라이제이션     | 한국어 + 영어 (MVP)                                     |
| 콘텐츠 번역           | 구매 시 번역 언어 선택 (태국어, 중국어 등) + 추가금     |
| 정산 출금             | 한국 작가: 원천징수 3.3% 후 계좌 입금, 월 1회           |
| PC 레이아웃           | 독자 영역: 모바일 고정, 내 서점 영역: PC 수동 전환 지원 |

## Feature 클러스터 (16개 PRD)

### Tier 1: Foundation (기반)

#### F1. 인증 & 계정 관리

- 소셜 로그인 (카카오/구글/네이버)
- 비밀번호 찾기/변경
- 프로필 설정
- 회원 탈퇴 (메일 요청)
- 성인 인증 (전화번호, 한국 유저)
- 예상 US: ~6개
- 구현 상태: 부분 구현 (이메일 로그인 존재)

#### F2. 서점 시스템

- 서점 오픈 (필명+서점명 최소 요건)
- 서점 페이지 (독자 공개 프로필)
- 서점 소개글/프로필 이미지 설정
- 서점 관리 메인 화면
- 예상 US: ~4개
- 구현 상태: 미구현

#### F15. 플랫폼 기반

- 약관/개인정보처리방침
- i18n (한국어+영어 UI)
- 성인 등급 시스템 (작가 등록)
- 앱 세팅
- 예상 US: ~4개
- 구현 상태: 미구현

### Tier 2: Content (콘텐츠 생산)

#### F3. 글쓰기 & 발행

- 글쓰기 에디터 (PC 레이아웃 지원)
- 단편/시리즈 화 선택
- 장르 태그 3계층 설정
- 접근 권한/가격 설정
- 발행/임시저장
- 예상 US: ~6개
- 구현 상태: 부분 구현 (CKEditor)

#### F4. 시리즈 관리

- 시리즈 생성 (기본 정보 입력)
- 연재 추가 (화차 자동 채번)
- 시리즈 완결 처리
- 연재 목록 관리
- 예상 US: ~4개
- 구현 상태: 미구현

### Tier 3: Discovery (콘텐츠 발견)

#### F5. 홈 & 탐색

- 홈화면 (에디터스픽 배너 + 장르 탭 + 인기/신작 + 최신 피드)
- 장르 태그 기반 탐색 (GL/BL/Hetero + 하위 태그)
- 에디터스픽 큐레이션
- 비로그인 접근 (SEO)
- 예상 US: ~6개
- 구현 상태: 부분 구현 (기본 피드 존재)

#### F6. 작품 상세 & 발견

- 단편 상세 페이지 (썸네일+제목+작가+태그+별점+미리보기+가격+구매)
- 시리즈 상세 페이지 (시리즈 요약+평점+연재 목록+편당 가격/구매)
- 미리보기 (freeContent)
- 공유 링크 (단편/시리즈편/시리즈목록)
- 북마크 (장바구니/관심 목록)
- 예상 US: ~5개
- 구현 상태: 부분 구현

#### F16. 검색 시스템

- 장르 태그 필터 검색
- 키워드 검색 (Elasticsearch)
- 커스텀 태그 매칭 (검색 결과에만 반영)
- 하위 장르 태그 브라우징
- 예상 US: ~4개
- 구현 상태: 미구현

### Tier 4: Monetization (수익화)

#### F7. 포인트 시스템

- 포인트 충전 (PG 연동, 다국적 결제 수단)
- 포인트 월렛 (잔액 관리)
- 포인트 사용 이력 조회
- 충전/차감 트랜잭션 관리
- 예상 US: ~4개
- 구현 상태: 미구현

#### F8. 구매 & 내 서재

- 작품 구매 (포인트 차감)
- 번역 언어 선택 (원문/영어/태국어/중국어 + 추가금)
- 추가 번역 구매
- 내 서재 담기 (무료: 바로 담기, 유료: 구매 시 자동 담기)
- 구매 작품 아카이브 (작가 삭제 시에도 유지)
- 중복 구매 방지
- 예상 US: ~6개
- 구현 상태: 미구현

#### F14. 번역 인프라

- 번역 API 연동
- 번역 결과 캐싱 (동일 번역 재사용)
- 제목/요약 자동 번역 (탐색용 무료)
- 번역 품질 피드백 (백오피스 → 운영진)
- 예상 US: ~4개
- 구현 상태: 미구현

### Tier 5: Reading (읽기 경험)

#### F9. 전자책 뷰어

- 전자책 뷰어 (PC + 모바일 반응형)
- 원문/번역 모드 전환 (디폴트: 번역본)
- 이어서 읽기 (진행률 저장, 자동 위치 기억)
- 페이지↔화차 매핑 (시리즈 연재 목록과 동기화)
- 뷰어 내 연재 목록 메뉴 + 바로 읽기
- 별점 남기기 (편당, 읽기 완료 후)
- 예상 US: ~5개
- 구현 상태: 미구현

### Tier 6: Engagement (참여)

#### F10. 소셜 인터랙션

- 팔로우 (서점 팔로우)
- 응원의 글 작성 (서점 페이지 노출)
- 응원 글 삭제/신고
- 찜하기
- 예상 US: ~4개
- 구현 상태: 부분 구현 (팔로우 존재)

#### F11. 알림 시스템

- 글 발행 시 팔로워 알림
- 작가 수정 알림 (구매자에게, 작가 선택)
- 알림 목록 조회
- 예상 US: ~3개
- 구현 상태: 미구현

### Tier 7: Analytics (운영)

#### F12. 작가 통계 대시보드

- 정산 탭: 이달 총 수익 / 작품별 수익 (단편 vs 시리즈, 시리즈는 편당 하위 목록)
- 독자 관리 탭: 팔로우 수, 유료 구매자 수(unique), 찜 수, 응원 수

## 구조 정의

### 내 서재 구조

```
내 서재
├── 무료 작품: 바로 담기 → 전자책 뷰어로 전체 읽기
├── 유료 단편: 담기(미리보기만) → 구매 시 자동 담기 → 전체 읽기
└── 유료 시리즈: 시리즈 통째로 담기 → 편당 구매 → 구매한 편 읽기
    └── 썸네일 클릭 → 전자책 뷰어 오픈 → 뷰어 내 메뉴에서 연재 목록
```

### 북마크 vs 내 서재

```
북마크 = 관심 표시 (장바구니 역할)
  → 북마크 목록에서 클릭 → 작품 상세 페이지로 이동

내 서재 = 실제 보관함
  → 썸네일 클릭 → 전자책 뷰어 직접 오픈
```

### 이어서 읽기 동기화

```
전자책 뷰어 380/500p (17화 중간)
    ↕ 동기화
서점 홈 > 시리즈 > 연재 목록:
    1~16화: 읽음
    17화: 읽는 중
    18화~: 안 읽음
    + "바로 읽기" 버튼 → 뷰어로 이동
```

### 이중 리더 (번역 모드)

```
전자책 뷰어 열기
    → 디폴트: 구매한 번역 언어로 표시
    → 모드 전환 버튼: "원문 보기" / "번역본 보기"
    → 나란히 보기 (동시 표시)는 MVP 이후 검토
```

### 단편 vs 시리즈 상세 페이지

```
단편 상세:
├── 썸네일 + 제목 + 작가(서점명) + 태그
├── 별점
├── 미리보기 (freeContent)
├── 가격 + 번역 선택 + 구매 버튼
└── 찜/북마크 버튼

시리즈 상세:
├── 시리즈 썸네일 + 제목 + 작가(서점명) + 태그
├── 시리즈 평점 (편별 평균)
├── 시리즈 요약
├── 번역 언어 선택 옵션
└── 연재 목록 (편당 가격 + 구매 버튼 + 미리보기)
```

### 통계 대시보드 MVP

```
작가 통계 대시보드 (월 단위 조정)
├── [정산 탭]
│   ├── 이달 총 수익
│   └── 작품별 수익
│       ├── 단편: 단건 수익
│       └── 시리즈: 전체 수익 + 편당 수익 (하위 목록)
├── [독자 관리 탭]
│   ├── 팔로우 수
│   ├── 유료 구매자 수 (unique)
│   ├── 찜 수
│   └── 응원 수
└── [콘텐츠 관리 탭]
    ├── 작품별 조회수
    ├── 미리보기 → 유료 구매 전환 수
    └── 작품 당 평균 별점
```

### PC 레이아웃 전략

```
독자 영역 (홈, 검색, 내 서재, 상세, 뷰어):
    → 모바일 UI 고정 (PC에서도 모바일처럼)

내 서점 영역 (서점 관리, 에디터, 대시보드):
    → 모바일 먼저 → 사용자 수동 "PC 전환" 버튼으로 PC 레이아웃
    → 자동 전환 안 됨
```
