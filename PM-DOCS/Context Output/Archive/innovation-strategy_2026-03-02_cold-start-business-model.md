## Executive Summary

Readly의 Cold Start Problem은 **"수요 창출"이 아니라 "기존 수요의 합법적 연결"**이다.

태국 GL 팬픽 시장에는 이미 200명 이상의 한국 작가가 활동하고, 일반 GL 대비 3~14배 조회수를 기록하며, 해외에서는 중국 웨이보 무단 번역이 일어날 정도로 수요가 존재한다. 물이 없는 것이 아니라, 수도관이 없는 것이다.

**핵심 전략: 이중 파괴(Dual Disruption)**

| 전략 축    | 유형                  | 핵심 논리                                     |
| ---------- | --------------------- | --------------------------------------------- |
| **주축**   | New-market Disruption | 갈증이 있는데 물이 없는 non-consumption 해소  |
| **보조축** | Low-end Disruption    | 트레이드오프 없는 저수수료(10%) + 글로벌 리치 |

**이중 브랜딩**: 팬픽 코어(covert) + GL/BL 플랫폼(public)으로 시장 진입하되, 복제 불가 자원(BD 매니저, 에디터스픽 큐레이션, 번역 인프라)으로 방어벽을 구축한다.

**Day 1 MVP 7개 기능**으로 0-3개월 내 Foundation을 완성하고, 핵심 가정 A1(BD 매니저 5명+ 작가 온보딩)을 최우선 검증한다.

## Phase 1: Strategic Context (기회 발견)

### 핵심 발견: Cold Start의 재정의

| 요소                  | 내용                                                              |
| --------------------- | ----------------------------------------------------------------- |
| **BD 매니저**         | 사내 이사, 10명 미만 핵심 작가와 신뢰 관계 보유, 본인도 팬픽 작가 |
| **단행본 실적**       | 인당 200권 판매 실적                                              |
| **해외 수요 근거**    | 번역 요청 + 중국 웨이보 무단 번역 발생                            |
| **Cold Start 재정의** | 수요 창출 X → **기존 수요의 합법적 연결**                         |

### 전략적 함의

Cold Start Problem의 본질은 "닭과 달걀 중 무엇이 먼저인가"가 아니다. 이미 닭(작가)도 달걀(독자)도 존재한다. 문제는 둘을 연결하는 **합법적이고 수익화 가능한 채널**이 없다는 것이다. BD 매니저가 이 연결의 첫 번째 파이프라인 역할을 한다.

### 3-2. BMC 핵심 인사이트

| 인사이트                        | 설명                                                            |
| ------------------------------- | --------------------------------------------------------------- |
| **번역 프리미엄: 비용 to 수익** | 번역 비용이 곧 수익 모델이 되는 구조 (탐색 무료 + 구매시 과금)  |
| **BD 매니저: 복제 불가 자원**   | 사내 이사 + 작가 본인 + 신뢰 관계 = 경쟁사가 채용으로 복제 불가 |
| **수수료 10%: 착한 Lock-in**    | 포스타입 대비 압도적 저수수료로 전환 비용이 아닌 잔류 인센티브  |

### 3-3. Value Proposition Canvas (VPC)

**작가 측**:

| Pain                                | Gain                           |
| ----------------------------------- | ------------------------------ |
| 전속 계약 종속                      | 계약 없이 자유 발행            |
| 해외 수요는 있지만 합법 수익화 불가 | 번역 인프라로 글로벌 수익 발생 |
| 수수료 30%+ 과다                    | 10% 이하 저수수료              |
| 독자 데이터 접근 불가               | 글로벌 리치 대시보드(차후)     |

**독자 측**:

| Pain                                  | Gain                         |
| ------------------------------------- | ---------------------------- |
| 좋아하는 작가 콘텐츠에 합법 접근 불가 | 작가가 있는 곳에서 합법 구매 |
| 언어 장벽                             | 실시간 번역으로 소비         |
| 무단 번역의 낮은 품질                 | 번역 프리미엄 고품질 번역    |
| 적응된 고통 (불편을 당연시)           | 적응된 고통 = 최대 기회      |

**VPC 핵심 인사이트**:

- 번역 = 양면 시장의 Bridge (작가와 독자를 동시에 연결하는 핵심 축)
- 작가 Pain 해소 → 독자 Gain 자동 생성 (Virtuous Cycle)
- 독자의 적응된 고통(무단 번역에 익숙) = 최대 기회

## Phase 4: Disruptive Opportunities (파괴적 포지셔닝)

### 4-1. 이중 파괴 전략 (Dual Disruption)

| 유형                  | 역할      | 핵심 논리                                                                                 |
| --------------------- | --------- | ----------------------------------------------------------------------------------------- |
| **New-market** (주축) | 시장 창출 | "갈증이 있는데 물이 없는 non-consumption" 해소. 해외 GL/BL 독자에게 합법적 유통 채널 제공 |
| **Low-end** (보조축)  | 경쟁 방어 | 트레이드오프 없는 저수수료(10%) + 글로벌 리치. 기존 플랫폼 대비 "더 싸면서 더 넓음"       |

이중 파괴는 동시에 이중 방어벽을 형성한다: New-market 측면에서 기존 플랫폼이 접근하지 않는 시장을 선점하고, Low-end 측면에서 수수료 경쟁력으로 작가 이탈을 방지한다.

### 4-2. JTBD (Jobs to Be Done)

**작가 Trigger**: "태국에서 340명이 읽었다" (글로벌 리치 대시보드의 핵심 감정)
**독자 Trigger**: "좋아하는 작가가 여기 있다" (작가 팔로우 = 플랫폼 선택)

**유입 시퀀스**: BD 매니저 → 작가 → 독자 (이 순서만이 유일한 시퀀스)

| Job 유형       | 작가                      | 독자                      |
| -------------- | ------------------------- | ------------------------- |
| **기능적 Job** | 글로벌 발행 + 수익화      | 합법적 콘텐츠 접근 + 번역 |
| **감정적 Job** | "내 글이 세계에서 읽힌다" | "덕질의 합법화"           |
| **사회적 Job** | 프로 작가로 인정          | 팬덤 커뮤니티 소속감      |

감정적 Job이 전환의 열쇠이다. 기능만으로는 플랫폼 전환 비용을 넘을 수 없다.

### 4-3. ERRC Grid

| 전략          | 요소                 | 설명                                               |
| ------------- | -------------------- | -------------------------------------------------- |
| **Eliminate** | 전속 계약            | 작가 자율성 극대화                                 |
|               | 지역 잠금            | 글로벌 동시 발행                                   |
|               | 알고리즘 경쟁        | 커플링태그 카탈로그 + 에디터스픽 큐레이션으로 대체 |
| **Reduce**    | 수수료 → 10%         | 포스타입 30%+ 대비                                 |
|               | 편집 부담            | 간소화된 발행 워크플로우                           |
|               | 검열                 | 작가 자율 등급 설정                                |
| **Raise**     | 수익 투명성          | 실시간 정산 + 글로벌 데이터(차후)                  |
|               | 작가 자율성          | 가격/권한/발행 일정 완전 통제                      |
|               | 번역 품질            | AI 번역 + 프리미엄 번역 옵션                       |
| **Create**    | 실시간 번역 인프라   | 탐색 무료 + 구매시 프리미엄                        |
|               | 글로벌 리치 대시보드 | 국가별 독자 현황(차후)                             |
|               | 번역 프리미엄 모델   | 비용 → 수익 전환                                   |
|               | BD 매니저 온보딩     | 복제 불가 채널                                     |
|               | 크로스보더 팬덤 연결 | 언어 장벽 없는 팬덤                                |

**MVP 핵심 결정**: 알고리즘 Eliminate → 커플링태그(GL/BL 광범위 장르 수준만) + 에디터스픽(사용자 본인 직접 운영) 큐레이션으로 대체.

**커플링명 직접 태그 불가**: 초상권 리스크, 아이돌 서치 방지 약어 관행, 태그 오염 문제로 자동 태그는 GL/BL 광범위 장르만 지원.

### 5-2. Value Chain 분석

```
생산(BD매니저) → 번역(핵심Moat) → 유통(GL/BL장르태그+에디터스픽) → 소비(이중리더) → 수익화(10%+글로벌정산)
```

| 단계       | 핵심 활동                    | Moat 요소                                      |
| ---------- | ---------------------------- | ---------------------------------------------- |
| **생산**   | BD 매니저 경유 작가 온보딩   | BD 매니저 = 복제 불가                          |
| **번역**   | AI 번역 + 프리미엄           | 핵심 경쟁력, 비용→수익 전환                    |
| **유통**   | GL/BL 장르 태그 + 에디터스픽 | 에디터스픽 = 사용자 본인 직접 운영 = 복제 불가 |
| **소비**   | 이중 리더 (국내+해외)        | 번역이 양면 시장 Bridge                        |
| **수익화** | 10% 수수료 + 글로벌 정산     | 착한 Lock-in                                   |

**자동 태그 범위**: GL/BL 광범위 장르만. 커플링명 직접 태그 불가 (초상권, 아이돌 서치 방지 약어, 태그 오염).

**취약 지점**: 경쟁사 해외 진출 / **완화**: 대규모 조직의 의사결정 속도 느림 = 선점 시간 버퍼

## Phase 6: Strategic Synthesis (전략 종합)

### 6-1. Strategic Narrative

```
기회 발견 → 시장 검증 → 모델 설계 → 파괴적 포지셔닝 → 실행 계획
```

1. **기회 발견**: BD 매니저라는 복제 불가 자원과, 이미 존재하는 해외 수요를 발견
2. **시장 검증**: $15-30M SAM, 직접 경쟁자 없는 Blue Ocean, Postype 벤치마크로 시장 실재 확인
3. **모델 설계**: 번역 비용을 수익으로 전환하는 BMC, 양면 시장을 번역이 Bridge하는 VPC
4. **파괴적 포지셔닝**: New-market(주축) + Low-end(보조) 이중 파괴, 감정적 Job이 전환 열쇠
5. **실행 계획**: Day 1 MVP 7개 기능, BD매니저→작가→독자 유일 시퀀스, 선점 속도 > 기능 완성도

### 6-2. Critical Assumptions (핵심 가정)
