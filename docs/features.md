# 기능 명세서

## 개요

Readly는 크리에이터와 독자를 연결하는 유료 콘텐츠 플랫폼입니다. 이 문서는 플랫폼의 모든 주요 기능을 상세히 설명합니다.

## 사용자 유형별 기능

### 1. 팔로워 (독자)

#### 1.1 계정 관리
- **회원가입/로그인**
  - 이메일 회원가입
  - 소셜 로그인 (Google, Kakao)
  - 이메일 인증
  - 비밀번호 찾기/재설정

- **프로필 관리**
  - 프로필 사진 업로드
  - 닉네임 설정
  - 자기소개 작성
  - 관심 카테고리 설정

#### 1.2 콘텐츠 탐색
- **홈 피드**
  - 인기 포스트
  - 최신 포스트
  - 추천 에디터
  - 카테고리별 필터링

- **검색 기능**
  - 포스트 제목/내용 검색
  - 에디터 검색
  - 태그 검색
  - 고급 검색 필터

- **카테고리**
  - 카테고리별 브라우징
  - 서브 카테고리 지원
  - 인기 카테고리 표시

#### 1.3 포스트 열람
- **무료 포스트**
  - 전체 내용 열람
  - 댓글 작성
  - 좋아요/북마크

- **유료 포스트**
  - 미리보기 (첫 20%)
  - 개별 구매
  - 구매 내역 확인

- **구독자 전용 포스트**
  - 구독 시 전체 열람
  - 구독 혜택 안내

#### 1.4 구독 시스템
- **에디터 구독**
  - 월간 구독
  - 구독 혜택 확인
  - 자동 갱신 설정
  - 구독 취소

- **구독 관리**
  - 구독 중인 에디터 목록
  - 구독 알림 설정
  - 구독 내역

#### 1.5 결제 시스템
- **결제 수단**
  - 신용/체크카드
  - 간편결제 (토스페이, 카카오페이)
  - 포인트 충전

- **결제 관리**
  - 결제 내역 조회
  - 영수증 발급
  - 환불 요청

#### 1.6 상호작용
- **댓글**
  - 댓글 작성/수정/삭제
  - 대댓글
  - 댓글 좋아요

- **소셜 기능**
  - 포스트 공유
  - 에디터 팔로우
  - 활동 피드

#### 1.7 알림
- **알림 유형**
  - 새 포스트 알림
  - 댓글 알림
  - 구독 만료 알림
  - 프로모션 알림

- **알림 설정**
  - 푸시 알림
  - 이메일 알림
  - 알림 카테고리별 설정

### 2. 에디터 (크리에이터)

#### 2.1 에디터 등록
- **에디터 신청**
  - 신청서 작성
  - 포트폴리오 제출
  - 심사 과정
  - 승인/거절 알림

- **에디터 프로필**
  - 상세 프로필 작성
  - 대표 이미지 설정
  - SNS 링크 연동
  - 카테고리 설정

#### 2.2 포스트 관리
- **포스트 작성**

  - 마크다운 지원
  - 이미지/동영상 업로드
  - 자동 저장
  - 미리보기

- **포스트 설정**

  - 공개 예약

- **포스트 관리**

  - 임시 저장 관리
  - 수정/삭제
  - 통계 확인

#### 2.3 구독자 관리
- **구독자 현황**
  - 총 구독자 수
  - 신규/이탈 구독자
  - 구독자 분석
  - 구독자 목록

- **구독 플랜**
  - 구독 가격 설정
  - 구독 혜택 설정
  - 특별 혜택 관리

#### 2.4 수익 관리
- **수익 대시보드**
  - 총 수익
  - 포스트별 수익
  - 구독 수익
  - 월별/연별 통계

- **정산**
  - 정산 신청
  - 정산 내역
  - 세금 관련 서류
  - 계좌 관리

#### 2.5 분석 도구
- **콘텐츠 분석**
  - 조회수/읽음 시간
  - 인기 포스트
  - 독자 인구 통계

- **수익 분석**
  - 수익 추이
  - 구매 전환율
  - 구독 유지율
  - 수익 예측

#### 2.6 커뮤니케이션

  - 공지사항 작성
  - 메시지 기능
  - 이벤트 진행

- **피드백**
  - 독자 피드백 수집
  - 설문조사
  - 만족도 조사

## 주요 기능 상세

### 포스트 접근 권한 시스템

#### 1. 전체 공개
- 모든 사용자 열람 가능
- 비로그인 사용자도 접근 가능
- SEO 최적화 적용
- 소셜 미디어 공유 가능

#### 2. 구독자 전용
- 해당 에디터 구독자만 열람
- 구독 유도 메시지 표시
- 구독 혜택 안내
- 부분 미리보기 제공

#### 3. 구매자 전용 (유료)
- 개별 구매 필요
- 가격 표시
- 구매 후 영구 열람
- 구매 전 미리보기 (20%)

#### 4. 비공개
- 에디터 본인만 열람
- 임시 저장 용도
- URL 공유 불가
- 검색 제외

### 결제 시스템 상세

#### 결제 플로우
1. **포스트 구매**
   ```
   선택 → 결제 수단 선택 → 결제 진행 → 완료 → 열람 권한 부여
   ```

2. **구독 결제**
   ```
   구독 선택 → 플랜 확인 → 결제 → 자동 갱신 설정 → 구독 활성화
   ```

#### 결제 수단
- **신용/체크카드**: Visa, MasterCard, JCB, UnionPay
- **간편결제**: 토스페이, 카카오페이, 네이버페이
- **포인트**: 플랫폼 내 포인트 시스템
- **쿠폰**: 프로모션 쿠폰, 할인 코드

#### 환불 정책
- **포스트 구매**: 열람 전 7일 이내 환불 가능
- **구독**: 남은 기간 일할 계산 환불
- **자동 환불**: 시스템 오류, 중복 결제
- **환불 불가**: 이미 소비된 콘텐츠

### 추천 시스템

#### 콘텐츠 추천
- **협업 필터링**: 유사 사용자의 행동 분석
- **콘텐츠 기반**: 읽은 포스트와 유사한 콘텐츠
- **인기도 기반**: 조회수, 구매수, 평점
- **개인화**: 관심 카테고리, 읽기 패턴

#### 에디터 추천
- **구독 패턴 분석**: 유사 구독자의 다른 구독
- **카테고리 매칭**: 관심 카테고리의 인기 에디터
- **신규 에디터**: 주목할 만한 신인 에디터
- **큐레이션**: 관리자 추천

### 알림 시스템

#### 실시간 알림
- **웹 푸시**: 브라우저 알림
- **인앱 알림**: 앱 내 알림 센터
- **배지**: 읽지 않은 알림 수

#### 알림 종류
1. **콘텐츠 알림**
   - 새 포스트 발행
   - 구독 에디터 활동
   - 관심 카테고리 업데이트

2. **상호작용 알림**
   - 댓글/답글
   - 좋아요
   - 팔로우

3. **시스템 알림**
   - 결제 완료/실패
   - 구독 만료 예정
   - 시스템 공지

### 분석 도구

#### 에디터 대시보드
- **실시간 통계**: 오늘의 조회수, 수익
- **트렌드 분석**: 성장률, 패턴 분석
- **독자 분석**: 연령, 성별, 지역, 관심사
- **수익 최적화**: A/B 테스트, 가격 추천

#### 관리자 대시보드
- **플랫폼 메트릭**: DAU, MAU, 거래액
- **콘텐츠 품질**: 평균 읽기 시간, 완독률
- **수익 분석**: 수수료 수익, 성장 추이
- **이상 감지**: 비정상 패턴, 어뷰징
