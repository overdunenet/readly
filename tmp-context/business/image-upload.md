# 이미지 업로드

## 목적

에디터가 포스트 작성 시 이미지를 삽입할 수 있도록 지원합니다. S3 기반 외부 저장소를 사용하여 안정적인 이미지 호스팅과 CDN 배포를 제공합니다.

## 핵심 기능

| 기능 | 설명 | 사용자 관점 |
| ---- | ---- | ----------- |
| 이미지 삽입 | 포스트 에디터에서 이미지 파일 선택 및 업로드 | 드래그앤드롭 또는 파일 선택으로 간편 삽입 |
| 이미지 교체 | 기존 이미지 블록에서 다른 이미지로 교체 | 이미지 클릭 후 교체 버튼으로 변경 |
| CDN 배포 | 업로드된 이미지를 CDN URL로 서빙 | 빠른 이미지 로딩 |

## 사용자 흐름

1. 에디터가 포스트 작성 화면에서 이미지 삽입 버튼 클릭 또는 파일 드래그
2. 이미지 파일 검증 (50MB 이하, 이미지 형식만 허용)
3. 서버에서 업로드 URL 발급 -> 이미지 저장소에 직접 업로드
4. 업로드 완료 후 CDN URL이 에디터에 자동 삽입
5. 포스트별 경로로 이미지가 정리되어 관리 용이

## 제약사항

| 항목 | 제한 |
| ---- | ---- |
| 파일 크기 | 최대 50MB |
| 파일 형식 | 이미지 파일만 (image/*) |
| 저장 경로 | 포스트별 분리 (post/postId/) |
