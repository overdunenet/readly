# GitHub Project 태스크 조회

다음 단계를 수행하세요:

## 1. 현재 태스크 조회

```bash
gh project item-list 4 --owner overdunenet --format json
```

## 2. 결과 분석 및 안내

조회된 태스크를 다음 기준으로 분류하여 보여주세요:

### 긴급 (P0)

- P0 태스크가 있으면 **즉시 처리 필요** 알림

### 진행 중 (In progress)

- 현재 진행 중인 태스크 표시
- 계속 진행할지 확인

### 대기 중 (Ready)

- 시작 가능한 태스크를 우선순위순으로 표시
- P1 → P2 순서

### 백로그 (Backlog)

- 아직 시작 전인 태스크 개수만 표시

## 3. 출력 형식

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
📋 Readly 태스크 현황
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🔴 긴급 (P0): [있음/없음]
- [태스크 제목]

🔵 진행 중:
- [태스크 제목]

📝 Ready (시작 가능):
1. [P1] 태스크 제목
2. [P2] 태스크 제목

📦 Backlog: N개

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
💡 다음 작업: [추천 태스크]
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 4. 다음 작업 추천

1. P0이 있으면 → P0 태스크 추천
2. In progress가 있으면 → 계속 진행 추천
3. Ready P1이 있으면 → P1 태스크 추천
4. 없으면 → Backlog에서 선택 제안
