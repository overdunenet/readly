# 인증 & 계정 관리

## 목적

글로벌 및 국내 사용자가 빠르고 안전하게 가입/로그인하고, 성인 콘텐츠에 접근하며, 계정 생애주기를 관리할 수 있도록 한다.

## 핵심 기능

| 기능 | 설명 | 사용자 관점 |
| --- | --- | --- |
| Passwordless 인증 | 소셜 로그인 + 전화번호 OTP | 비밀번호 없이 빠른 가입/로그인 |
| 국가별 Provider 분기 | 한국(카카오/네이버/Google), 글로벌(Google/Facebook/Line) | 자국 익숙한 소셜 계정으로 가입 |
| 성인인증 | 한국: 통신사 본인확인, 글로벌: 여권 OCR | 성인 콘텐츠 접근 허용 |
| 회원 탈퇴 | 30일 유예기간, 재로그인 시 복구 | 실수 방지, 안전한 탈퇴 |

## 사용자 흐름

1. 사용자가 로그인 화면에서 국가(한국/글로벌) 선택 → 해당 소셜 Provider 표시
2. 소셜 버튼 클릭 또는 전화번호 입력 → OAuth/OTP 인증 완료
3. 최초 가입 시 닉네임 설정 → 가입 완료
4. 성인 콘텐츠 접근 시 성인인증 안내 → 국가별 인증 수행
5. 탈퇴 요청 → 즉시 로그아웃 → 30일 내 재로그인 시 복구 가능

## 비즈니스 정책

### OTP 제한

- 유효시간: 3분
- 재발송 쿨다운: 60초
- 시간당 5회, 일일 10회 (전화번호 기준)
- 최대 인증 시도: 5회/건

### 성인인증 정책

- 한국: 만 19세 이상 (NICE/KCB 통신사 본인확인)
- 글로벌: 만 20세 이상 (AWS Textract 여권 OCR)
- 유효기간: 1년 (만료 시 재인증)
- 유효기간 내 재인증 불가

### 탈퇴 정책

- 요청 시 즉시 로그아웃
- 30일 유예기간 중 재로그인 시 '탈퇴 철회' 동의 팝업 → 복구
- 30일 경과: 계정 영구 삭제, 포스트는 "탈퇴한 사용자"로 익명 변환
- 영구 삭제 대상: 계정 정보, 소셜 연동, 팔로우 관계

### 계정 병합

- 같은 이메일로 여러 소셜 Provider 가입 시 이메일 기준 자동 병합
