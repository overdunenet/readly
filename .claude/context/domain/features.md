# 기능 명세

## 사용자 유형

### 1. 팔로워 (독자)

- 콘텐츠 소비자
- 포스트 구매/구독

### 2. 에디터 (크리에이터)

- 콘텐츠 생산자
- 포스트 작성/수익 관리

### 3. 관리자

- 플랫폼 운영자
- 사용자/콘텐츠 관리

## 포스트 접근 권한 시스템

| 권한         | 설명        | freeContent 접근            | paidContent 접근            |
| ------------ | ----------- | --------------------------- | --------------------------- |
| `public`     | 전체공개    | 모든 사용자 (비로그인 포함) | 모든 사용자 (비로그인 포함) |
| `subscriber` | 구독자 전용 | 모든 사용자 (비로그인 포함) | 해당 에디터 구독자만        |
| `purchaser`  | 구매자 전용 | 모든 사용자 (비로그인 포함) | 개별 구매자만               |
| `private`    | 비공개      | 에디터 본인만               | 에디터 본인만               |

> `freeContent`는 `private`를 제외한 모든 접근 권한에서 공개됩니다.
> `paidContent`는 접근 권한에 따라 결제/구독 여부를 확인한 후 노출됩니다.

## 유료/무료 섹션 구분

Post는 **freeContent**(무료 공개 구간)와 **paidContent**(유료 구간) 두 개 필드로 구성됩니다.

| 구분      | 필드          | 용도                                    |
| --------- | ------------- | --------------------------------------- |
| 무료 구간 | `freeContent` | 미리보기 역할. 항상 노출 (private 제외) |
| 유료 구간 | `paidContent` | 본문 콘텐츠. 결제/구독 후 노출          |

### 에디터 작성 시

- 에디터가 Post 작성 시 무료 구간과 유료 구간을 **별도로 작성**
- `freeContent`만 작성 시: 전체 무료 Post (유료 구간 없음)
- `paidContent`까지 작성 시: 부분 유료 Post

### 팔로워 열람 시

- 무료 구간(`freeContent`) 먼저 표시
- 유료 구간이 존재하면 잠금 표시 + 결제/구독 유도
- 결제/구독 완료 후 전체 콘텐츠 열람

> 상세 구현은 `domain/post.md` 참조

## 콘텐츠 타입

| contentType | 설명                             | 현재 상태          |
| ----------- | -------------------------------- | ------------------ |
| `text`      | 장문 텍스트 (팬픽, 소설, 에세이) | **구현 중**        |
| `image`     | 이미지 기반 (웹툰, 일러스트)     | 미구현 (확장 예정) |
| `mixed`     | 텍스트 + 이미지 혼합             | 미구현 (확장 예정) |

## 주요 기능

### 팔로워 기능

- 회원가입/로그인 (이메일, 소셜)
- 포스트 열람/검색
- 무료 구간 미리보기
- 포스트 구매 (유료 구간 열람)
- 에디터 구독 (월간)
- 댓글/좋아요/북마크

### 에디터 기능

- 포스트 작성 (리치 텍스트 에디터 - CKEditor 5)
- **무료/유료 구간 분리 작성** (freeContent / paidContent)
- 접근 권한/가격 설정
- 구독자 관리
- 수익 대시보드
- 분석 도구

### 관리자 기능

- 회원 관리
- 콘텐츠 모더레이션
- 결제/정산 관리
- 시스템 설정

## 결제 시스템

### 결제 수단

- 신용/체크카드
- 간편결제 (토스페이, 카카오페이)

### 결제 유형

1. **포스트 구매**: 개별 포스트의 유료 구간(paidContent) 열람권 구매
2. **구독 결제**: 월간 자동 갱신, 에디터의 구독자 전용 Post 전체 열람

### 환불 정책

- 포스트: 열람 전 7일 이내
- 구독: 남은 기간 일할 계산
