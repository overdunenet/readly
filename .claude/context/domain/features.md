# 기능 명세

## 사용자 유형

### 1. 팔로워 (독자)

- 콘텐츠 소비자
- 포스트 구매/구독

### 2. 에디터 (크리에이터)

- 콘텐츠 생산자
- 포스트 작성/수익 관리

### 3. 관리자

- 플랫폼 운영자
- 사용자/콘텐츠 관리

## 포스트 접근 권한 시스템

| 권한         | 설명        | 접근 가능                   |
| ------------ | ----------- | --------------------------- |
| `public`     | 전체공개    | 모든 사용자 (비로그인 포함) |
| `subscriber` | 구독자 전용 | 해당 에디터 구독자만        |
| `purchaser`  | 구매자 전용 | 개별 구매자만               |
| `private`    | 비공개      | 에디터 본인만               |

## 주요 기능

### 팔로워 기능

- 회원가입/로그인 (이메일, 소셜)
- 포스트 열람/검색
- 포스트 구매
- 에디터 구독 (월간)
- 댓글/좋아요/북마크

### 에디터 기능

- 포스트 작성 (리치 텍스트 에디터)
- 접근 권한/가격 설정
- 구독자 관리
- 수익 대시보드
- 분석 도구

### 관리자 기능

- 회원 관리
- 콘텐츠 모더레이션
- 결제/정산 관리
- 시스템 설정

## 결제 시스템

### 결제 수단

- 신용/체크카드
- 간편결제 (토스페이, 카카오페이)

### 결제 유형

1. **포스트 구매**: 개별 포스트 구매
2. **구독 결제**: 월간 자동 갱신

### 환불 정책

- 포스트: 열람 전 7일 이내
- 구독: 남은 기간 일할 계산
